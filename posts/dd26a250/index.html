Ubuntu 18.04 一键安装深度截图工具 Deepin Screenshot

发表于 2019-06-08 更新于 2023-04-08 分类于 os ， ubuntu 阅读次数： 本文字数： 73 阅读时长 ≈ 1 分钟

Deepin Screenshot，一个简洁易用的截图工具。

Ubuntu 18.04 一键安装深度截图工具 Deepin Screenshot

安装

  • 打开 Ubuntu Software
  • 搜索 deepin
  • 点击 Deepin Screenshot
  • 点击 Install

添加快捷键

  • 打开 Settings
  • 选择 Devices，选择 Keyboard
  • 滑到底部选择添加
